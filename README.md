# Webpack boilerplate for React with TypeScript

Webpack with babel (jsx), sass (scss), postcss (autoprefixer, css-nano, mqpacker and etc) and TypeScript loader.

## Installation

```javascript
$ npm i
```

or

```javascript
$ yarn
```

## Usage

### Open development server:

```javascript
$ npm start
```

or

```javascript
$ yarn start
```

### Generate production build:

```javascript
$ npm run build
```

or

```javascript
$ yarn build
```
